# Практическое задание: Интерактивность и События (ЛР9)

## Цель задания

Научиться перехватывать действия пользователя (клики, ввод текста, отправку форм) и программно изменять состояние интерфейса без перезагрузки страницы, строго соблюдая архитектуру разделения JS и CSS.

## Техническое задание

Вся логика пишется в вашем файле `script.js`.

### Шаг 1: Бургер-меню или Модальное окно (Событие `click`)

Реализуйте открытие и закрытие всплывающего окна (или мобильного меню) по клику.

1. В HTML создайте кнопку открытия (например, `<button class="btn-open">Открыть</button>`) и сам блок окна (`<div class="modal">...</div>`). Внутри окна добавьте кнопку закрытия.
2. В CSS скройте окно по умолчанию (например, через `opacity: 0` и `pointer-events: none`) и создайте класс `.active`, который делает окно видимым.
3. В JavaScript найдите обе кнопки и само окно с помощью `querySelector`.
4. Повесьте на кнопку открытия слушатель `addEventListener('click', ...)`, который будет **только добавлять** класс `.active` вашему окну через метод `classList.add()`.
5. Повесьте на кнопку закрытия слушатель, который будет **удалять** этот класс через `classList.remove()`.

### Шаг 2: Перехват отправки формы (Метод `preventDefault`)

Добавьте на сайт форму обратной связи, которая не должна перезагружать страницу при отправке.

1. Создайте в HTML тег `<form class="contact-form">` с парой полей и кнопкой `type="submit"`.
2. В JS найдите эту форму и подпишитесь на событие `'submit'` (или подпишитесь на `'click'` по кнопке отправки).
3. Первой же строчкой внутри вашей функции-обработчика остановите стандартное поведение браузера с помощью объекта события (`event`).
4. Выведите в `console.log()` сообщение: *"Форма перехвачена, перезагрузки не было!"*.

### Шаг 3: «Живой» счетчик символов (Событие `input`)

Сделаем интерактивное поле ввода, как в Twitter (ограничение на длину сообщения).

1. Добавьте в форму `<textarea class="message-input"></textarea>` и под ней пустой тег `<p class="char-counter">0/100</p>`.
2. В JS подпишитесь на событие `'input'` для этого текстового поля.
3. Внутри функции получайте текущий текст из поля (подсказка: свойство `event.target.value`).
4. Измеряйте длину этого текста и динамически перезаписывайте содержимое счетчика (`textContent`), чтобы при каждом нажатии клавиши цифра обновлялась (например, "15/100", "16/100").
