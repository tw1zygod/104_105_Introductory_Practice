# Практическое задание: Динамический рендеринг интерфейса (ЛР8)

## Цель задания

Освоить программную генерацию HTML-разметки. Научиться связывать локальную базу данных (массив объектов) с визуальным интерфейсом через методы DOM API (`querySelector`, `createElement`, `textContent`, `append`).

## Техническое задание

Вам предстоит полностью перевести блок «Мое портфолио» (или аналогичный блок с карточками) на вашем сайте на динамический рендеринг.

### Шаг 1. Очистка статики (Подготовка HTML)

Ваш HTML должен стать чистым и модульным.

1. Откройте ваш `index.html` и найдите секцию с карточками ваших проектов/товаров.
2. **Удалите** весь HTML-код самих карточек.
3. Оставьте только пустой родительский контейнер и убедитесь, что у него есть понятный класс (например, `<div class="projects-gallery"></div>`). На странице этот блок теперь должен выглядеть абсолютно пустым.

### Шаг 2. Инициализация в JavaScript

Откройте ваш подключенный файл `script.js`.

1. Скопируйте туда ваш массив `projects` (базу данных) из задания 7-го дня. Убедитесь, что в нем есть минимум 4-5 объектов с названиями, категориями и другими данными.
2. Используя метод `document.querySelector()`, найдите ваш пустой контейнер `.projects-gallery` и сохраните ссылку на него в константу (например, `const galleryContainer`).

### Шаг 3. Фабрика компонентов (Цикл рендеринга)

Вам нужно написать алгоритм, который автоматически построит карточки.

1. Запустите метод `forEach()` для вашего массива `projects`.
2. **Внутри цикла (для каждого проекта) выполните следующие строгие шаги:**
* **Создание:** С помощью `document.createElement()` создайте главный `div` для карточки, а также внутренние теги (например, `h3` для заголовка, `p` для категории).
* **Стилизация:** Добавьте главному `div` карточки нужный CSS-класс через свойство `.classList.add()`, чтобы к ней применились ваши стили из CSS-файла.
* **Наполнение:** Используя свойство `.textContent`, аккуратно перенесите данные из текущего объекта (название, категорию) внутрь созданных тегов (`h3`, `p`).
* **Сборка:** Используя метод `.append()`, вложите текстовые теги (`h3`, `p`) внутрь `div` карточки.
* **Публикация:** В самом конце цикла используйте `.append()`, чтобы вложить готовую карточку внутрь глобального `galleryContainer`.

### Критерии успешной сдачи:

1. Если открыть ваш `index.html`, там нет ни одной жестко прописанной карточки.
2. На самом сайте карточки отображаются корректно, стили не сломались.
3. Если добавить в массив в JavaScript еще один объект, на сайте автоматически (после обновления страницы) появляется новая карточка.
