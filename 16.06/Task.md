# Практическое задание: Семантический каркас страницы-визитки

## Цель задания

Создать базовую личную страницу-визитку на чистом HTML без CSS, чтобы отработать структуру документа, корректное использование тегов и основы семантики.

## Техническое задание

### Шаг 1. Базовая структура и SEO

1. Создайте директорию `MyPortfolio` в корне проекта и внутри нее файл `index.html`.
2. Добавьте базовую структуру HTML5: `<!DOCTYPE html>`, `html`, `head`, `body`.
3. В блоке `head` укажите корректную кодировку UTF-8.
4. Заполните тег `title`: укажите Имя и Фамилию.
5. Добавьте мета-тег `description` с коротким описанием страницы.

### Шаг 2. Шапка сайта

1. Используйте тег `header`.
2. Внутри разместите главный заголовок страницы `h1` (на странице он должен быть только один).
3. Добавьте блок навигации `nav`.
4. Внутри `nav` создайте маркированный список `ul` из трех ссылок:
	- Обо мне
	- Мое хобби
	- Контакты
5. На данном этапе для ссылок можно использовать `href="#"`.

### Шаг 3. Главный контент

1. Оберните основной контент страницы в тег `main`.
2. Создайте первую секцию `section` с заголовком `h2` «Обо мне».
3. Добавьте в эту секцию 2-3 абзаца `p` с рассказом о себе.
4. Добавьте изображение через тег `img`.
5. Обязательно заполните атрибут `alt` у изображения.
6. Создайте вторую секцию `section` «Мое хобби».
7. Внутри второй секции создайте независимый смысловой блок `article` и опишите в нем одно хобби.

### Шаг 4. Боковая панель и футер

1. Добавьте боковую панель `aside`.
2. В `aside` разместите нумерованный список `ol` из трех пунктов (например, любимые книги или фильмы).
3. Добавьте футер `footer`.
4. В `footer` укажите:
	- знак копирайта (©),
	- текущий год,
	- ссылку на гитхаб.
5. Ссылка в футере должна открываться в новой вкладке (`target="_blank"`).

## Требования к выполнению

1. Выполнять задание только на HTML, без CSS.
2. Использовать семантические теги по назначению.
3. Соблюдать иерархию заголовков (`h1` -> `h2` -> `h3` при необходимости).
4. Не использовать `div` там, где есть подходящий семантический тег.
